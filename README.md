# example.com

An about me web site.

## Dependencies

* PHP 7.2 or higher
* Composer
* An API key for Mailgun

## Installation

Clone this project into a web directory
```sh
cd /var/www
git clone git@github.com:amenahussaini/example.com.git

cd /var/www/example.com
composter install
```
